# 🗺️ UNO Online — Roadmap

## ✅ v1.0 — MVP (Current)
- [x] Real-time multiplayer (2-10 players)
- [x] Room creation with shareable invite links
- [x] Full UNO rules (Skip, Reverse, Draw 2, Wild, Wild Draw 4)
- [x] UNO button + penalty system
- [x] Turn timer with auto-draw
- [x] In-game chat with emoji reactions
- [x] Bot takeover for disconnected players
- [x] Reconnection support (3-minute window)
- [x] Server-authoritative validation
- [x] Responsive mobile design
- [x] Deployed on Render (free tier)

---

## 🔜 v1.1 — Polish & UX
- [ ] Sound effects (card play, UNO call, turn change, win)
- [ ] Card play animations (fly to discard pile)

- [ ] Vibration feedback on mobile

- [ ] Loading skeletons instead of spinners

## 🎨 v1.2 — Customization
- [ ] Player avatars / profile pictures

## 🏆 v1.3 — Competitive
- [ ] Win/loss statistics (localStorage)

- [ ] Player ratings / ELO system

- [ ] Game replay / match history

## 🎲 v1.4 — Advanced Rules
- [ ] Stacking +2 / +4 cards
- [ ] 7-0 swap rule (optional)

- [ ] Challenge Wild Draw 4
- [ ] Custom house rules per room
- [ ] Different game modes (Speed UNO, Teams)

## 🤖 v1.5 — AI & Bots
- [ ] Bot difficulty levels (Easy / Medium / Hard)
- [ ] Practice mode (play vs bots offline)
- [ ] AI opponent using game theory
- [ ] Bot personalities (aggressive, defensive, random)

## 📱 v2.0 — Platform
- [ ] PWA (installable on phone home screen)
- [ ] Push notifications for game invites
- [ ] Voice chat integration
- [ ] Tournament / bracket system
- [ ] Clubs / friend lists
- [ ] Multi-language support (i18n)

## 💰 Monetization Ideas
- [ ] Non-intrusive banner ads (lobby only)
